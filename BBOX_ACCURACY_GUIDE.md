# 检测框位置和大小优化指南

## ✅ 已改进的功能

### 1. 更准确的边界框计算
- 使用凸包算法计算边界
- 根据模板长宽比调整检测框
- 自动添加适当边距

### 2. 改进的平滑算法
- 使用加权平均（置信度高的权重大）
- 增加平滑窗口到 8 帧
- 减少抖动和跳跃

### 3. 更好的可视化
- 加粗边界框（4像素）
- 显示中心十字线
- 显示尺寸信息（宽x高）
- 更大更清晰的标签

## 📊 检测效果评估

### 好的检测
```
✅ 边界框紧贴物体
✅ 尺寸与实际相符
✅ 位置稳定（不抖动）
✅ 置信度 > 60%
✅ 匹配点 > 20
```

### 需要改进的检测
```
⚠️  边界框太大或太小
⚠️  位置偏移
⚠️  抖动明显
⚠️  置信度 < 40%
⚠️  匹配点 < 10
```

## 🔧 调试技巧

### 1. 查看匹配点数

检测框下方显示：
```
Matches: 50
```

- **> 30** - 很好，检测可靠
- **20-30** - 良好
- **10-20** - 一般
- **< 10** - 不可靠

### 2. 查看置信度

标签显示：
```
yazi 89%
```

- **> 70%** - 绿色框，高置信度
- **40-70%** - 黄色框，中等置信度
- **< 40%** - 橙色框，低置信度

### 3. 查看尺寸

检测框下方显示：
```
187x174px
```

对比训练样本的平均尺寸（187x174），如果差异很大说明检测不准确。

## 💡 常见问题

### Q1: 检测框比实际物体大

**原因：**
- 匹配点分布太散
- 包含了背景特征

**解决：**
- 重新标注，框选更准确
- 使用纹理更明显的物体
- 改善光照，减少阴影

### Q2: 检测框比实际物体小

**原因：**
- 匹配点集中在物体中心
- 边缘特征不明显

**解决：**
- 确保物体边缘清晰
- 增加训练样本
- 包含更多角度

### Q3: 检测框位置偏移

**原因：**
- 角度差异
- 距离差异
- 光照差异

**解决：**
- 保持与训练时相似的角度
- 保持相似的距离
- 保持相似的光照

### Q4: 检测框抖动

**原因：**
- 匹配点不稳定
- 特征不够明显

**解决：**
- 已启用平滑算法（8帧）
- 如果还抖动，增加平滑窗口
- 或者使用更稳定的物体
