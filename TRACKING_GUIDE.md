# 动态跟踪工具使用指南

## 🎯 功能特性

### 核心功能
- ✅ **手动标记** - 鼠标拖动框选物体
- ✅ **自动跟踪** - 标记后自动跟踪物体运动
- ✅ **轨迹显示** - 实时显示运动轨迹
- ✅ **多目标跟踪** - 同时跟踪多个物体
- ✅ **跟踪丢失检测** - 自动检测并移除丢失的目标

### 跟踪器类型
- **CSRT** - 精确但慢，适合复杂场景
- **KCF** - 速度和精度平衡，推荐日常使用
- **MOSSE** - 快速但不精确，适合实时性要求高的场景

## 🚀 启动方式

```bash
cd ~/Desktop/industrial_action_recognition

# 使用默认跟踪器（CSRT）
./run_tracking.sh

# 使用快速跟踪器（MOSSE）
./run_tracking.sh --tracker MOSSE

# 使用平衡跟踪器（KCF）
./run_tracking.sh --tracker KCF

# 指定摄像头
./run_tracking.sh --camera 1

# 指定输出目录
./run_tracking.sh --output my_tracking_data
```

## ⌨️ 操作说明

### 标记与跟踪
1. **鼠标左键拖动** - 在物体周围绘制矩形框
2. 松开鼠标 - 自动开始跟踪该物体
3. 物体会被实时跟踪，显示边界框和运动轨迹

### 类别切换
- **数字键 1-9** - 切换标注类别
- 标记前选择类别，不同类别用不同颜色显示

### 跟踪控制
- **SPACE（空格）** - 暂停/继续跟踪
  - 暂停时可以添加新的跟踪目标
  - 继续后所有目标恢复跟踪
- **C** - 清除所有跟踪目标
- **D** - 删除最后添加的跟踪目标

### 显示选项
- **T** - 切换轨迹显示（开/关）
- **L** - 切换标签显示（开/关）
- **R** - 切换跟踪器类型（CSRT → KCF → MOSSE → CSRT）

### 保存与退出
- **S** - 保存当前帧和跟踪数据
- **H** - 显示帮助信息
- **Q** - 退出程序

## 📊 显示信息

### 边界框
- **颜色** - 根据物体类别自动分配
- **粗细** - 根据跟踪置信度调整
  - 粗线 = 高置信度 (>70%)
  - 细线 = 低置信度 (<70%)

### 标签信息
```
#1 welding 85%
```
- `#1` - 跟踪对象编号
- `welding` - 物体类别
- `85%` - 跟踪置信度

### 轨迹显示
- **彩色线条** - 物体运动轨迹（最近30帧）
- **小圆点** - 历史位置点
- **大圆点** - 当前中心位置

### 状态栏
- 当前选择的类别
- 正在跟踪的物体数量
- 当前帧数
- 显示选项状态

## 🎯 最佳实践

### 标记技巧
1. **紧贴边缘** - 框选时尽量贴近物体边缘
2. **完整包含** - 确保物体完全在框内
3. **避免遮挡** - 在物体清晰可见时标记
4. **合适大小** - 框不要太大或太小

### 跟踪优化
1. **光照稳定** - 避免剧烈光照变化
2. **背景简洁** - 减少背景干扰
3. **运动平滑** - 避免物体突然消失或快速移动
4. **及时清理** - 删除跟踪失败的目标

### 性能调优
- **快速场景** - 使用 MOSSE 跟踪器
- **精确场景** - 使用 CSRT 跟踪器
- **多目标** - 使用 KCF 平衡性能
- **减少目标** - 同时跟踪不超过 5-10 个物体

## 🔧 故障排除

**跟踪丢失？**
- 物体移动太快 → 使用 MOSSE 跟踪器
- 遮挡严重 → 重新标记
- 光照变化 → 改善照明条件

**性能卡顿？**
- 跟踪目标太多 → 删除不需要的目标
- 跟踪器太慢 → 切换到 KCF 或 MOSSE
- 关闭轨迹显示 → 按 T 键

**标记不准确？**
- 暂停画面 → 按 SPACE
- 仔细框选 → 确保边界准确
- 删除重做 → 按 D 删除后重新标记
