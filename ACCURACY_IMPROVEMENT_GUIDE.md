# 检测位置和大小准确性改进

## 🎯 问题

检测框的位置和大小与标注的物体不一致。

## ✅ 解决方案

已创建**精确检测器**，使用改进的算法：

### 1. 凸包计算
- 使用匹配点的凸包计算边界
- 比简单的最小/最大值更准确

### 2. 长宽比保持
- 根据训练模板的长宽比调整检测框
- 避免检测框变形

### 3. 加权平滑
- 使用置信度作为权重
- 高置信度的检测影响更大
- 边界框更稳定

### 4. 尺寸显示
- 在检测框下方显示精确尺寸
- 便于验证准确性

## ⚙️ 技术细节

### 边界框计算流程

1. **特征匹配** - 找到对应的特征点
2. **凸包计算** - 计算匹配点的凸包
3. **最小外接矩形** - 获取旋转矩形
4. **长宽比调整** - 根据模板调整比例
5. **加权平滑** - 使用历史数据平滑

### 平滑算法

```python
# 加权平均
weighted_x = Σ(x_i * confidence_i) / Σ(confidence_i)
weighted_y = Σ(y_i * confidence_i) / Σ(confidence_i)
weighted_w = Σ(w_i * confidence_i) / Σ(confidence_i)
weighted_h = Σ(h_i * confidence_i) / Σ(confidence_i)
```

## 🐛 常见问题

### Q: 检测框还是不准确？

**A:** 可能原因：
1. 训练数据标注不准确
2. 物体角度差异太大
3. 光照条件不同
4. 物体尺寸变化大

**解决：**
- 检查 `training_samples_preview/`
- 重新录制更准确的数据
- 保持环境一致

### Q: 检测框抖动？

**A:** 已使用加权平滑，如果还抖动：
- 增加平滑窗口（修改 `smooth_window`）
- 提高检测置信度阈值
- 改善光照稳定性

### Q: 尺寸总是偏大/偏小？

**A:** 
- 检查训练时的标注是否准确
- 调整边距参数
- 使用更多训练样本
