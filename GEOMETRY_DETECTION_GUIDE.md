# 物体几何信息检测

## 📐 新增功能

现在检测器可以获取物体的详细几何信息：

### 1. 旋转角度
- 物体相对于水平方向的旋转角度
- 范围：-90° 到 +90°
- 用紫色箭头显示方向

### 2. 尺寸信息
- **宽度**：物体的宽度（像素）
- **高度**：物体的高度（像素）
- **面积**：物体的面积（平方像素）

### 3. 形状信息
- **长宽比**：宽度/高度的比值
- **中心点**：物体的中心坐标
- **旋转矩形**：最小外接旋转矩形

## 🎨 可视化显示

### 边界框
- **绿色矩形** - 轴对齐边界框
- **紫色矩形** - 旋转矩形（更精确）
- **蓝色轮廓** - 物体精确轮廓

### 中心点和方向
- **圆点** - 物体中心
- **紫色箭头** - 指示旋转角度和方向

### 文字信息
```
Angle: 45.2°        # 旋转角度
Size: 200x150       # 宽度 x 高度
Area: 30000px²      # 面积
Ratio: 1.33         # 长宽比
```

## 💡 应用场景

### 1. 质量检测
- 检查零件尺寸是否符合规格
- 验证长宽比是否正确
- 测量面积

### 2. 姿态估计
- 检测物体的旋转角度
- 判断物体是否摆放正确
- 机器人抓取定位

### 3. 分类识别
- 根据长宽比区分不同类型
- 根据面积过滤大小
- 根据角度判断方向

### 4. 统计分析
- 记录物体尺寸分布
- 分析旋转角度范围
- 计算平均面积

## 📐 几何信息说明

### 旋转角度
- **0°** - 水平方向
- **正值** - 逆时针旋转
- **负值** - 顺时针旋转
- **±90°** - 垂直方向

### 长宽比
- **1.0** - 正方形
- **> 1.0** - 横向矩形
- **< 1.0** - 纵向矩形

### 面积
- 单位：平方像素
- 可用于过滤大小
- 可用于质量检测

## 🔧 实际应用示例

### 示例 1：检测螺栓方向
```python
# 读取检测结果
with open('detection_result.json') as f:
    data = json.load(f)

for det in data['detections']:
    if det['label'] == '螺栓':
        angle = det['geometry']['angle']
        
        if abs(angle) < 5:
            print("✅ 螺栓方向正确")
        else:
            print(f"⚠️  螺栓旋转了 {angle:.1f}°")
```

### 示例 2：尺寸检测
```python
for det in data['detections']:
    width = det['geometry']['width']
    height = det['geometry']['height']
    
    # 检查尺寸规格
    if 190 <= width <= 210 and 190 <= height <= 210:
        print("✅ 尺寸合格")
    else:
        print(f"❌ 尺寸不合格: {width:.0f}x{height:.0f}")
```

### 示例 3：形状分类
```python
for det in data['detections']:
    ratio = det['geometry']['aspect_ratio']
    
    if ratio > 2.0:
        print("长条形物体")
    elif ratio < 0.5:
        print("高瘦形物体")
    else:
        print("方形物体")
```

## 📊 精度说明

### 角度精度
- 理论精度：±0.1°
- 实际精度：±2-5°（取决于特征点质量）

### 尺寸精度
- 像素级精度
- 受摄像头分辨率影响
- 需要标定才能转换为实际尺寸

### 提高精度的方法
1. 使用高分辨率摄像头
2. 固定摄像头距离
3. 标注更多样本
4. 改善光照条件
5. 进行相机标定

## 🎯 相机标定（可选）

如果需要测量实际尺寸（毫米、厘米），需要进行相机标定：

```python
# 标定比例（像素/毫米）
pixel_per_mm = 10  # 需要实际测量

# 转换为实际尺寸
width_mm = width_pixels / pixel_per_mm
height_mm = height_pixels / pixel_per_mm
```

## 🔍 调试技巧

### 查看旋转矩形
- 紫色矩形显示最小外接矩形
- 比绿色矩形更精确
- 可以看到物体的真实方向

### 查看方向箭头
- 箭头指向物体的"长边"方向
- 箭头长度固定为 50 像素
- 用于快速判断方向

### 切换显示
- 按 G 键可以关闭几何信息
- 减少屏幕杂乱
- 提高性能
