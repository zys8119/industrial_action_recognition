# 智能标注工具 - 提高标注准确性

## 🎯 问题解决

如果你觉得手动标注不准确，这个智能标注工具提供了三大辅助功能：

### 1. 🧲 边缘吸附（Edge Snap）
- **问题**：鼠标绘制的框边缘不准确
- **解决**：自动检测物体边缘，鼠标会自动吸附到最近的边缘
- **效果**：框选更精确，边界更贴合物体

### 2. 🔧 自动调整（Auto Adjust）
- **问题**：手动框选的范围不够精确
- **解决**：使用图像分割算法自动优化边界框
- **效果**：自动收缩或扩展框，紧贴物体轮廓

### 3. 💡 智能建议（Suggestions）
- **问题**：不知道哪里有物体需要标注
- **解决**：自动检测画面中的显著区域
- **效果**：灰色虚线框提示可能的目标位置

## 🚀 使用方法

```bash
cd ~/Desktop/industrial_action_recognition

# 启动智能标注工具
./run_smart_annotator.sh

# 指定摄像头
./run_smart_annotator.sh --camera 1
```

## ⌨️ 操作说明

### 基本标注
1. 用鼠标拖动绘制矩形框
2. 松开鼠标，工具会自动优化边界
3. 按空格保存当前帧的所有标注

### 切换辅助功能
- **E 键** - 开启/关闭边缘吸附
- **A 键** - 开启/关闭自动调整
- **S 键** - 开启/关闭智能建议

### 其他操作
- **数字键 1-9** - 切换类别
- **C 键** - 清除当前所有标注
- **U 键** - 撤销最后一个标注
- **Q 键** - 退出

## 💡 使用技巧

### 提高边缘吸附效果
1. **充足光照** - 确保物体边缘清晰
2. **简洁背景** - 减少背景干扰
3. **慢速移动** - 鼠标移动慢一点，吸附更准确

### 提高自动调整效果
1. **粗略框选** - 大致框住物体即可
2. **包含完整** - 确保物体完全在框内
3. **避免遮挡** - 物体不要被遮挡

### 使用智能建议
1. **观察灰框** - 灰色虚线框是系统建议
2. **参考位置** - 在建议位置附近绘制
3. **调整大小** - 根据建议调整框的大小

## 🎨 界面说明

### 视觉提示
- **彩色边缘** - 启用边缘吸附时显示
- **灰色虚框** - 智能建议的候选区域
- **彩色实框** - 已确认的标注

### 状态栏
- **Label** - 当前选择的类别
- **Annotations** - 当前帧的标注数量

## 🔧 故障排除

### 边缘吸附不工作？
- 按 E 键确认功能已开启
- 检查物体边缘是否清晰
- 尝试调整光照条件

### 自动调整效果不好？
- 按 A 键确认功能已开启
- 确保框选包含完整物体
- 避免复杂背景

### 智能建议太多/太少？
- 按 S 键切换功能
- 调整摄像头角度
- 改善光照条件

### 标注框跳动？
- 关闭边缘吸附（按 E）
- 使用更稳定的支架
- 减少画面晃动
