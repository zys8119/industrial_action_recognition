# 完整工作流程 - 从标注到识别

## 🎯 三步完成物体识别

### 步骤 1️⃣：标注物体
```bash
cd ~/Desktop/industrial_action_recognition
./run_smart_annotator.sh
```

**操作：**
1. 用鼠标框选要识别的物体
2. 按数字键选择类别
3. 按**空格键保存**（重要！）
4. 重复标注多个样本（建议每类 5-10 个）
5. 按 Q 退出

**标注技巧：**
- 每个物体至少标注 5 个不同角度/位置的样本
- 标注要准确，框要紧贴物体边缘
- 开启边缘吸附（E键）和自动调整（A键）

### 步骤 2️⃣：自动识别
```bash
./run_detector.sh
```

**效果：**
- 系统自动加载你标注的数据
- 学习物体的特征
- 在摄像头画面中实时检测相同的物体

### 步骤 3️⃣：查看结果
- 绿色框 = 高置信度检测
- 黄色框 = 中等置信度
- 橙色框 = 低置信度
- 显示匹配点数量和置信度

./run_smart_annotator.sh

```bash

```

## 🔍 工作原理

### 标注阶段
1. 你手动框选物体
2. 系统提取框内的图像
3. 提取 SIFT 特征点
4. 保存为模板

### 识别阶段
1. 读取所有标注的模板
2. 在摄像头画面中提取特征
3. 与模板进行特征匹配
4. 找到相似的物体并标注

## 🔧 常见问题

### Q: 标注后检测不到？
**A:** 检查以下几点：
1. 确认按了空格键保存标注
2. 检查 `data/smart_annotations/` 目录是否有文件
3. 标注的物体是否有足够的纹理特征
4. 尝试标注更多样本（5-10个）

### Q: 误检太多？
**A:** 提高检测阈值：
```bash
./run_detector.sh --min-matches 15 --confidence 0.7
```

### Q: 检测不到物体？
**A:** 降低检测阈值：
```bash
./run_detector.sh --min-matches 5 --confidence 0.4
```

### Q: 识别速度慢？
**A:** 
- 减少标注样本数量
- 降低摄像头分辨率
- 增加检测间隔（修改代码中的 `frame_count % 3`）

### Q: 如何添加新类别？
**A:**
1. 编辑 `configs/label_list.txt` 添加新类别
2. 运行标注工具，选择新类别标注
3. 重新运行检测器

## 🎯 适用场景

### ✅ 适合
- 固定形状的物体（螺栓、零件、产品）
- 纹理丰富的物体
- 小批量多品种检测
- 快速原型验证
- 无需大量训练数据

### ❌ 不适合
- 形变大的物体（布料、液体）
- 纹理单一的物体（纯色表面）
- 需要语义理解的场景
- 大规模生产线（建议用深度学习）
